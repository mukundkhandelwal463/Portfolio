# Mukund Khandelwal | Portfolio

Welcome to my portfolio! This repository contains my personal portfolio website, featuring a dark mode and a light mode.

## 🚀 LIVE DEMO
[Soon to be Hosted on GitHub Pages]

## 🛠️ Built With
- **HTML5**
- **Tailwind CSS** (Styling)
- **GSAP** (Animations)
- **Lenis** (Smooth Scrolling)
- **JavaScript** (Theme Toggling & Interactions)

## 📁 Project Structure
- `index.html`: Main dark-mode portfolio.
- `light.html`: Premium light-mode version.
- `assets/`: Contains project images, QR codes, and profile photo.
- `mukund cv.pdf`: My latest CV.
- `style.css`: Custom CSS styles.

## 🌟 Highlights
- **Dynamic Projects**: Interactive project cards with custom-designed badges.
- **Glassmorphism Design**: Modern, sleek UI with premium aesthetics.
- **Micro-animations**: Smooth transitions and hover effects using GSAP.
- **Dual Theme**: Seamless toggle between White/Green and Dark/Purple aesthetics.
